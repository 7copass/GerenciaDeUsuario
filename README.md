<h1>Gerencia de usuários</h1>

 
#### Projeto desenvolvido utilizando as tecnologias:
+ Java
+ SpringBoot
+ SpringSecurity
+ Thymeleaf
+ Materialize
+ Postgres

## Orientações:
+ tela de login
+ cadastro de usuário
+ edição de usuário
+ exclusão de usuário
+ busca por nome
+ inclusão de endereço de usuário, consumindo web service Via cep
+ inclusão de telefone de usuário
